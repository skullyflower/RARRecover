### Spot Check inventory forms

Users select from a list of programs, fill out the form, then get congratulated, and then presented with a summary they can copy and send to their sponsor's or whoever. 
The program specific lists of steps, and lists of responses, are stored in files with a *-constants.ts nameing convention stored in the directory with page file that consumes it.
